# Overview

Planbot is designed as an infrastructure to generate a human readable document base on a set of input and prompts.

The first application to generate proposals in private banking areas.

# Private banking proposal

Most proposal in private banking can be divided in data and prompts of the following areas.

## Data
- Client Profile 
    - Demographic
    - Risk rating
- Product catalogs
    - Product and their historical performance
    - Product rating across factors - expected return, certainty of return for a specified time horizon
- Common financial needs

## Prompts
- Guidelines for the tasks below
    - matching produce and client 
    - analysis pros and cons for the suggested vs existing
    - perform scenario analysis
- Output (Proposal) format
- Risk disclaimer
